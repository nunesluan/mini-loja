# 03-tailwind

Implementação da tela usando **Tailwind CSS**.

### Recursos:
- Navbar fixa com toggle de tema e badge do carrinho
- Grid de produtos responsivo (1–4 colunas)
- Card com hover, foco, skeleton
- Dark mode com persistência
- Acessibilidade: navegação por teclado, contraste ≥ 4.5:1
- Transições suaves (200ms)

### Rodar
```bash
npm install
npm run dev
